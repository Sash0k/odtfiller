Файл формата ODT представляет собой ZIP-архив. В этом архиве стили, просматриваемые через F11, хранятся в styles.xml, а код текста документа - в content.xml

Краткая справка по odtfiller может быть получена запуском программы в командной строке без параметров.

Создать стиль
Указать этот стиль в теге абзаца.

При сборке одного файла из нескольких шаблонов у них должен совпадать styles.xml. Для этого шаблоны должны быть созданы разделением одного файла.

При необходимости вставки кода:
- Создать в OpenOffice то, что надо вставлять.
- Найти код в content.xml, в том числе используемые стили
- Разместить код в программу


Поля создаются через Ctrl+F2 (Вставка/Поля/Дополнительно) вкладка "Переменные" тип поля - "поле пользователя".
